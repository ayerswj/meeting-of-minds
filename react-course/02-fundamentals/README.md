# 02. React Fundamentals

## Learning Objectives
- Understand the basics of React and component-based architecture
- Learn JSX syntax and rendering
- Manage state and props
- Handle events and render lists

## Topics Covered
- What is JSX?
- Rendering Elements
- Function vs Class Components
- Props and State
- Handling Events
- Conditional Rendering
- Lists and Keys
- Basic Styling (CSS, CSS-in-JS)
